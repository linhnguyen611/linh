Accessibility

BUBBA website provides information about hairdresser for everyone (include people of all abilities and disabilities) and elderly. For people who cannot see very well, BUBBA website provides the function which can increase the text size bigger, this will help people can see the text easier and read in a number of ways. To help people can see clearly all the information about BUBBA, the website makes the content of BUBBA covered in black background include the big text size with white colour (the contrast colours between black and white will make the text can be seen easier) will help people can see the text clearly, this also makes the page become more stylish. BUBBA website has another function which can change the colour of text from white to red for who want to see the information with different colour, by moving the mouse to the text where you want to see and the colour of the text will change to red, this will help people who do not want to see the information in white text but different colour. BUBBA also has another function which is to change the colour of navigation when people move the mouse to the link they want to access. This will attract to people and also it helps people to find out the page they want to go easier.

Usability

To become a successful website, BUBBA needs to make customers trust and visit the website often. To do that, the website needs to be consistent and easy to navigate. With good navigation, customers can search directly for the things that they want to look for easier. BUBBA website has the navigation menu which help customers can click on one of the buttons on navigation bar to move to the page that they want to look. This will make the visitors like BUBBA website and make them visit more often. However, BUBBA website provides English language and it is inconvenient for customers from other countries who do not understand what the website are talking about because they are not good at English, so they will decide to leave the website. This is why the website should have more than one language, this will help customers can select the language that they can understand clearly, from that they can decide to visit the website more often. BUBBA website has an excellent house style. The content of BUBBA website covered in black background. The header is within a black background to emphasise BUBBA title, include the white big text size also will help customers can see the text clearly, this also make the page become more stylish. The images that the website used in every web page are clear and easy to look. The website also informs customers about the new offers or new services in the home page which can be used to attract customers.
